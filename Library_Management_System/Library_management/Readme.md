# Library Management System

## Overview
This is a web-based Library Management System developed using **Java Servlet**, **Spring MVC**, **JSP**, and a **MySQL** database. The system aims to streamline library operations by providing an easy-to-use platform for students and librarians.

## Features
- Separate user panels for **Students** and **Librarians**.
- Students can browse, search, and request books.
- Librarians can manage book inventories, process book lending and returns, and handle user registrations.
- Role-based access control to ensure proper permissions.
- Efficient management of book records and user activities.

## Technologies Used
- **Backend:** Java Servlet, Spring MVC framework
- **Frontend:** JSP (JavaServer Pages)
- **Database:** MySQL (localhost setup)
- **Architecture:** Model-View-Controller (MVC)

## Installation & Setup
1. Clone the repository to your local machine.
2. Set up MySQL database locally (localhost).
3. Create the necessary database schema and tables as per the provided SQL scripts.
4. Configure the database connection settings in the project config files.
5. Deploy the application on a servlet container such as Apache Tomcat.
6. Access the application through a web browser via `http://localhost:8080/your-app-context`.

## Usage
- Students and librarians can log in through their respective panels.
- Students can view available books, search by categories, and request loans.
- Librarians can add, update, and delete book records, and manage issued books and returns.

## Notes
*Note: This project was developed using a local (localhost) MySQL database for testing and development purposes.*
